# Auto Voice Lines Edit and Naming
The "Auto Voice Lines Edit and Naming" tool simplifies the editing and recognition of voice lines in a single audio file via CSV import. Instead of manually splitting lines recorded in a continuous take by an actor, the CSV file provides line details that the tool then automatically transcribes using speech-to-text, subsequently editing and organizing the lines into distinct takes. In Reaper, the tool layers different versions of each line and appropriately renames regions, streamlining the overall process of managing voice lines efficiently.

Please note, since this tool is still in early beta, it may have some limitations or bugs. Feel free to join the Discord channel for support, feedback, and updates.
