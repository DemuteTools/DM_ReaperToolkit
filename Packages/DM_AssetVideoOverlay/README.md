# Asset video overlay
Asset to Video Overlay is a REAPER script that displays the names of currently playing items directly on your video. 

## How to Use:
- Adjust the **settings** to change how the overlay dext is displayed(color, size, position, background)
- **Apply to track**: Set the track name to the track you want to affect. The tool will apply a video processor to each media items on that track and its child tracks, 
wich will make the name of the media items overlay on the current video.
- **Apply to selected**: Select the Items you want to affect.  The tool will apply a video processor to each selected media item, 
wich will make the name of the media items overlay on the current video.
- **Clear Track**: Removes video processors on media items on the the set track(or its childeren)
- **Clear Selected**: Removes video processors on selected media items
